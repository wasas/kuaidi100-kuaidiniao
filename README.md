# KuaiDi100-KuaiDiNiao

PHP Wrapper for KuaiDiNiao.
快递100与快递鸟 物流公司编码互相转换


## Installing

1. require library

```shell
composer require xu42/kuaidi100-kuaidiniao
```

## Usage

1. getNiao

```php
LogisticCode::getNiao('yunda');
```

2. get100

```php
LogisticCode::get100('SF');
```
